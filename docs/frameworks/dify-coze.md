## Dify

### 简介

**Dify** 是一个开源的 LLM 应用开发平台，提供可视化的工作流编排和 RAG 引擎。

### 核心功能

```
┌─────────────────────────────────────────────┐
│              Dify 功能全景                    │
│                                             │
│  📝 Prompt IDE      → 可视化提示词编排       │
│  🔄 Workflow        → 工作流编排             │
│  📚 RAG Engine      → 知识库管理             │
│  🤖 Agent           → 智能体构建             │
│  🔌 Plugin          → 插件扩展               │
│  📊 Monitoring      → 使用监控               │
│  🚀 API             → API 部署               │
└─────────────────────────────────────────────┘
```

### 快速开始

```bash
# Docker 部署
git clone https://github.com/langgenius/dify.git
cd dify/docker
docker compose up -d

# 访问 http://localhost/install 进行初始化
```

### 使用流程

1. **创建应用** — 选择应用类型（聊天助手、Agent、工作流等）
2. **配置模型** — 选择 LLM 提供商和模型
3. **编写提示词** — 使用可视化编辑器编写 System Prompt
4. **添加工具** — 集成外部工具和 API
5. **上传知识库** — 上传文档构建 RAG
6. **测试发布** — 测试后发布为 API 或 Web 应用

### 工作流示例

```
用户输入
    ↓
[LLM 节点] 意图识别
    ↓
[条件分支]
├── 查询类 → [RAG 节点] 知识库检索 → [LLM 节点] 生成回答
├── 操作类 → [API 节点] 调用外部 API → [LLM 节点] 确认结果
└── 闲聊类 → [LLM 节点] 自由对话
    ↓
输出结果
```

## Coze（扣子）

### 简介

**Coze**（扣子）是字节跳动推出的 AI Bot 开发平台，门槛极低，适合快速搭建对话机器人。

### 核心功能

```
┌─────────────────────────────────────────────┐
│              Coze 功能全景                    │
│                                             │
│  🤖 Bot 创建       → 快速创建机器人          │
│  🔌 插件市场       → 丰富的插件生态          │
│  📚 知识库         → 文档/网页知识库         │
│  🔄 工作流         → 可视化工作流            │
│  🎯 发布渠道       → 微信/飞书/Discord 等    │
│  📊 数据分析       → 使用数据分析            │
└─────────────────────────────────────────────┘
```

### 快速开始

1. 访问 [coze.cn](https://www.coze.cn)（国内版）或 [coze.com](https://www.coze.com)（国际版）
2. 注册账号
3. 创建 Bot
4. 配置提示词和插件
5. 测试并发布

### 插件示例

```json
{
  "name": "weather",
  "description": "查询天气",
  "parameters": {
    "city": {
      "type": "string",
      "description": "城市名称"
    }
  }
}
```

## Dify vs Coze 选型指南

### 选择 Dify 如果你需要：

- ✅ 私有化部署，数据安全
- ✅ 开源，可定制
- ✅ 企业级功能
- ✅ 与现有系统集成
- ✅ 自定义模型

### 选择 Coze 如果你需要：

- ✅ 最快上手
- ✅ 丰富的插件生态
- ✅ 一键发布到社交平台
- ✅ 免费的模型额度
- ✅ 面向 C 端用户

## 适用场景

✅ **推荐使用：**
- 非技术人员搭建 Agent
- 快速原型验证
- 企业内部工具
- C 端对话机器人
- 不想写代码

⚠️ **不太适合：**
- 需要高度定制
- 复杂的 Agent 逻辑
- 对性能要求极高
