## 💡 学习建议

::: tip 🎯 学习路径建议
1. **入门**：先看 DeepLearning.AI 的免费课程，建立整体认知
2. **实践**：选择一个框架（推荐 LangChain），跟着官方教程动手做
3. **深入**：阅读框架源码，理解底层原理
4. **扩展**：学习 MCP/A2A 等协议，了解行业趋势
5. **项目**：构建自己的 Agent 项目，积累实战经验
:::

::: warning ⚡ 保持更新
AI Agent 领域发展极快，建议：
- 关注各框架的 GitHub Release
- 订阅 LangChain、OpenAI、Anthropic 的博客
- 加入 Discord/Reddit 社区参与讨论
- 定期查看 GitHub Trending 了解新项目
:::
